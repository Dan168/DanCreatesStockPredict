Algorithms used

Linear Regression

Think of it like drawing a straight line on a graph that best fits the data points. Linear regression allows us to estimate how a change in the independent variable influences the dependent variable. By analyzing this relationship, we can make predictions or draw conclusions about future data points.

Polynomial Regression

Similar to Linear Regression, imagine you have data points that don't follow a straight line pattern, but rather curve or bend in a certain way. Polynomial regression helps us model these curved relationships by using mathematical equations with higher-order terms (such as squared or cubed terms) of the independent variable.

Think of LSTM as a specialized "memory unit" within the neural network. It can selectively remember or forget information based on the input data and the context it has learned. This memory cell allows LSTM to retain important information and discard irrelevant or redundant information from previous time steps.

Neural Networks

Coming soon

A neural network is a computer system inspired by the structure and functioning of the human brain. It is designed to learn and make predictions or decisions from data.

Just like our brains have interconnected neurons, a neural network consists of artificial neurons called "nodes" or "neurons." These neurons are organized into layers, with each layer connected to the next, forming a network.
